# Whisper Field Deployment Guide

## Quick Start:
1. Push this code to GitHub
2. Create MySQL database on Render
3. Deploy backend service
4. Update frontend URL
5. Open frontend.html

## API Endpoints:
- GET /api/whispers?topic=all
- GET /api/whispers/:id
- POST /api/whispers
- GET /api/whispers/:id/replies
- POST /api/whispers/:id/replies
